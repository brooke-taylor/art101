Lab 11 - JavaScript Events and Forms

Challenge

We were tasked with experimenting in JavaScript events, through the use of buttons that alter user input [and even characteristics about the site!]

Problems

On multiple occassions, my functions weren't working properly, or buttons not outputting results as expected. Many of these errors were the result of names not matching up between classes and IDs from the js to html file - it got a bit hard to keep track of them all with how many functions I wound up incorporating in this lab. It's a good reminder of the case sensitive nature of coding!

Reflection

Despite the fair chunk of work in this lab, I really enjoyed my experience with it! Nothing was frustratingly challenging, rather it was a lot of small tweaking [and constantly refreshing the browser] to get everything working as intended. Task X was very challenging, but rewarding.
